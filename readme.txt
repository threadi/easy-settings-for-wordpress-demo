This repository contains a WordPress demo plugin for [Easy settings for WordPress](https://github.com/threadi/easy-settings-for-wordpress). It is intended to show the possibilities of the plugin. It is not intended to be used actively in a productive system. You are welcome to use the programming as a template for your own use of Easy settings for WordPress.
